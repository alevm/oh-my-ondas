**1. Individual Track Control:**

**2. Quick Mutes:**
```

  - Tap: Toggle mute/unmute
  - Hold: Solo (mutes all others)
  - Double-tap: Delete track

Visual feedback:
  - Dim LED = Muted
  - Bright LED = Active
  - Pulsing LED = Soloed
```

**3. DJ-Style Crossfader:**
```

  â†’ Crossfades between two source groups

  - Processed layers (T3, T4)

```

```

  â†’ Morphs between scenes
```

**5. Master Section:**
```
SHIFT + ENC 4 â†’ Master level
  â†’ Controls final output
  â†’ Safety limiter engaged
  â†’ Prevents clipping

SHIFT + MODE â†’ Master FX:
  - Compressor (glue)
```

**6. Sends Control:**
```
ENC 2 â†’ Select FX send (1 or 2)
ENC 3 â†’ Adjust send amount

  - Track 1: 20%
  - Track 3: 80%
  - Track 6: 40%

  - Track 2: 60%
  - Track 5: 30%

```

**7. Frequency Mixing:**
```
Hold SHIFT + Turn ENC 1:
  â†’ EQ sweep per track

Selected: Track 1 (Radio)
  - Turn left: Low-pass filter
    (removes highs, leaves bass)
  - Turn right: High-pass filter
    (removes lows, leaves treble)
  - Center: Flat (no filtering)

Creative use:
  - Bass frequencies: Contact mics
  - Mid frequencies: Vocals/synth
  - High frequencies: Ambience
  - Result: Clear, separated mix
```
