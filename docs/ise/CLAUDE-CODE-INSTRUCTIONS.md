## 2.4 Technical Specifications

```
AUDIO
- Sample Rate: 44.1 kHz
- Bit Depth: 24-bit
- Latency: <10ms
- Channels: 8 simultaneous sources
- Effects: 8 types (delay, granular, glitch, bitcrush, stutter, reverb, filter, tape)

SEQUENCER
- Tracks: 8
- Steps: 4-64 per pattern
- Patterns: 8 (A-H)
- Scenes: 4 with crossfade morphing
- Trig Conditions: ALL, PROB, FILL, !FILL, NTH, NEIGH
- Parameter Locks: Pitch, Slice, Filter, Decay per step

LOCATION
- GPS Accuracy: 5 meters
- Timestamp: ISO 8601
- Metadata: JSON export
- Verification: Hash of location+time+audio

PLATFORM
- Web Audio API (browser-based)
- No installation required
- Desktop and mobile
- Offline capable (PWA)

HARDWARE (prototype)
- Teensy 4.1 @ 600MHz
- ESP32 for WiFi/GPS
- 8 touch pads
- 4 encoders
- OLED display
- 3500mAh battery (5-7h)
```
