## üí° PRO TIPS

### For Recording:

1. **Practice the performance** - Do dry runs until smooth
2. **Record in segments** - Easier to edit than one long take
3. **Show your hands** - Helps viewers understand workflow
4. **Use a timer** - Visual proof of speed claims
5. **Keep it moving** - Quick cuts, don't let it drag

### For Voiceover:

1. **Conversational tone** - Not a lecture
2. **Emphasize numbers** - "FOUR point eight seconds"
3. **Pause for effect** - Let visuals breathe
4. **Match energy** - Excited when showing cool features

### For Editing:

1. **Cut out dead air** - Keep it tight
2. **Use text overlays** - Reinforce voiceover
3. **Add subtle music** - Don't overpower demo audio
4. **Show comparisons side-by-side** - Visual proof
5. **End with strong CTA** - Tell viewers what to do next
