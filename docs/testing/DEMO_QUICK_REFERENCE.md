# Oh My Ondas - Demo Quick Reference

## 60-Second Demo Script

### Opening (0-10s)
- **Visual:** Oh My Ondas loaded, ready to go
- **Action:** Click pads 1, 2, 3 → Press Space
- **Message:** "Create a beat in 4.8 seconds. 10x faster than hardware."

### Sequencer (10-25s)
- Show 8 tracks with patterns
- Click steps to toggle
- Change pattern (A→B→C)
- **Message:** "Professional sequencer with parameter locks"

### Punch Effects (25-40s)
- Start playback
- Hold Q (stutter), release
- Hold W (reverse), release
- Hold E (filter), release
- Hold T (tape), release
- **Message:** "Live performance effects - hold key to apply"

### Scenes (40-50s)
- Click Scene B → instant change
- Click Scene C → instant change
- **Message:** "Scene-based performance, instant switching"

### Close (50-60s)
- Show stats overlay:
  - 10x faster workflow
  - 80% Octatrack parity
  - 100% free
- **Message:** "Oh My Ondas. The fastest music sequencer. Free."
