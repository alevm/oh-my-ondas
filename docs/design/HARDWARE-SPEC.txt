DESIGN PHILOSOPHY
================================================================================
- Touch display handles: Mixer, Synth params, FX params, Scene/Pattern selection
- Physical controls for: Performance pads, real-time knobs, transport
- Similar to: Novation Circuit, OP-Z, Synthstrom Deluge approach
- Cost target: ~$280-300

================================================================================
PHYSICAL SPECIFICATIONS
================================================================================
Enclosure:      200 x 140 x 45mm aluminum (realistic, portable)
Weight:         ~600g with battery
Display:        5" IPS Touch (800x480) - shows SEQ + touch mixer/menus
Touch Pads:     8x capacitive (MPR121 + copper pads)
Encoders:       4x rotary with push button (context-sensitive)
Buttons:        3x transport (Play/Stop/Rec)
LEDs:           4x RGB (status/feedback)
Audio:          Stereo out (3.5mm), Mic in, Headphone out
Power:          3500mAh LiPo, USB-C charging

================================================================================
CONTROL MAPPING
================================================================================

PHYSICAL CONTROLS:
------------------
8 PADS:         Trigger samples/drums, velocity sensitive
4 ENCODERS:     Context-sensitive (changes based on display mode)
                - Default: VOL / PAN / FILTER / FX-SEND
                - Synth mode: ATK / DEC / SUS / REL
                - FX mode: DLY / REV / CRUSH / PARAM
3 TRANSPORT:    Play/Pause, Stop, Record

TOUCH DISPLAY (5"):
-------------------
- Sequencer grid (tap to toggle steps)
- Mixer faders (touch sliders for MIC/SMP/SYN/RAD/OUT)
- Pattern select (A-H buttons)
- Scene select (A-D buttons)
- Synth controls (waveform, filter, envelope visualization)
- FX parameters (sliders)
- Settings menu
- GPS/Location display

================================================================================
BUILD NOTES
================================================================================

1. Touch pads: Cut 8x copper tape squares (40x40mm), connect to MPR121
2. Display: Use SPI connection to Teensy, touch via I2C
3. Encoders: Use interrupts for smooth response
4. Audio: I2S to Audio Shield, line out via op-amp buffer
5. Power: TP4056 charges battery, boost to 5V, 3.3V regulator for logic
